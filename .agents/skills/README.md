# OpenCode Skills (Local)

Este directorio contiene skills locales del proyecto para guiar implementaciones repetibles.

## Skills disponibles

- `diseno-viga-dashboard-dark/skill.md`: guia de estilo dashboard premium (manteniendo la paleta actual del proyecto).
- `diseno-viga-formularios-tecnicos/skill.md`: guia para formularios numericos tecnicos con validaciones y resultados.

## Convenciones

- Ver `/.opencode/skills/CONVENTIONS.md` para naming, versionado y alcance.

## Nota

Estas skills quedan versionadas en el repo para que el equipo las reutilice. Si tu entorno OpenCode soporta carga de skills locales, podras invocarlas por nombre.
